# 使用mod修改mod
## 实用模组

查看和修改NBT：IBE editor  
物品下查看物品标签：Tag Tooltips  
物品下查看物品NBT：NBT Tooltip  
查看玩家当前属性：Apothic Attributes  
聊天栏指令自动填充NBT：NBT Autocomplete  
加载全局数据包和材质包：Global Packs  
生物随机属性：Mobs Properties Randomness  
控制：In Control!  
生物控制：Mob Control  
残忍的boss：Brutal Bosses  
村民配置：VillagerConfig  
多人boss：Multiplayer Bosses
