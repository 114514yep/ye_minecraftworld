# 整合包
```
整合包.zip
{
    manifest.json
    modlist.html
    overrides/
}
```
## manifest.json
包含CurseForge的项目和文件ID，启动器会读取并据此下载mod
## modlist.html
可有可无，方便人查看有哪些mod，安装器几乎不用。
## overrides
文件夹，**整合包的核心**。一般有如下文件
```
    config/             配置文件
    defaultconfigs/     默认世界配置
    world/datapacks/    数据包
    mods/               私有模组

    resourcepacks/      资源包
    scripts/            脚本
```
### mods/
凡是能在overrides中实现的，都应避免更改其他作者的mod，可以写一个独立的附属mod放入其中。
### scripts/
使用KubeJS制作的脚本集合，如果和datapacks/数据包重复，会按照脚本执行。
### config/和defaultconfig/
世界的基础配置和魔改
