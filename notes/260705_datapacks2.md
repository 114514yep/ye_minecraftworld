## 数据包识别文件pack.mcmeta ##
"pack":{}的大括号中可以添加对数据包的描述如兼容版本。  
将子包文件夹加入主包中需要在.mcmeta中使用**overlays**加入对其的描述。
```
"overlays":{
    "entries":[
        {
                #这个大括号可以视为子包的"pack":{}
        },
        {
                #并列的其他子包同理
        }
    ]
}
```
过滤某些数据文件需要使用**filter**
```
"filter":{
    "block":[
        {
            "namespace":"",    #通过命名空间屏蔽
            "path":""          #通过文件路径屏蔽
        }
    ]
}
```
